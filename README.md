# infinity-partner

The partner portal for salons, barbershops, clinics, and independent beauty professionals.

## Jornada do Usuário

| Etapa | Escopo |
|-------|--------|
| Descoberta | Landing page + chatbot IA |
| Onboarding | Cadastro rápido, setup automático |
| Agendamento | IA sugere horários e confirmações |
| Atendimento | Registro de serviços e insumos |
| Pagamento | Pix/cartão, insights financeiros |
| Pós-atendimento | Feedback + campanhas automáticas |
| Relatórios | Dashboard com métricas e recomendações |
| Fidelização | Promoções personalizadas e pontos |

## Plano de desenvolvimento

As tasks acionáveis de cada etapa estão documentadas em [tasks/README.md](tasks/README.md).
