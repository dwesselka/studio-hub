# Badge

Componente de etiqueta para status, categorias, contadores.

**Arquivo:** `src/ui-components/ui/badge.tsx`

## Variants

| Variant       | Uso          |
| ------------- | ------------ |
| `default`     | Padrão       |
| `secondary`   | Secundário   |
| `destructive` | Alerta/erro  |
| `outline`     | Borda apenas |

```tsx
<Badge>Ativo</Badge>
<Badge variant="destructive">Inativo</Badge>
```
