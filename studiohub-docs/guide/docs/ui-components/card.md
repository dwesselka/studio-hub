# Card

Componente de cartão container.

**Arquivo:** `src/ui-components/ui/card.tsx`

## Estrutura

```tsx
<Card>
  <CardHeader>
    <CardTitle>Título</CardTitle>
    <CardDescription>Descrição opcional</CardDescription>
  </CardHeader>
  <CardContent>Conteúdo principal</CardContent>
  <CardFooter>Ações do card (botões)</CardFooter>
</Card>
```

## Subcomponentes

| Componente        | Função                              |
| ----------------- | ----------------------------------- |
| `Card`            | Container com borda, fundo e sombra |
| `CardHeader`      | Cabeçalho (padding, flex)           |
| `CardTitle`       | Título do card                      |
| `CardDescription` | Descrição secundária                |
| `CardContent`     | Conteúdo principal                  |
| `CardFooter`      | Rodapé para ações                   |
