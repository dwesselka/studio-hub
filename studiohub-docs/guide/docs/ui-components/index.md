# 🎨 Catálogo de Componentes UI

## Objetivo

Apresentar a biblioteca de componentes reutilizáveis da interface do StudioHub baseada em Tailwind CSS e Radix UI primitives.

## Componentes Disponíveis

- **[Button](/ui-components/button)** — Botões interativos com múltiplas variantes e estados.
- **[Input](/ui-components/input)** — Campos de entrada de texto e formulários.
- **[Card](/ui-components/card)** — Contêineres de conteúdo visual.
- **[Badge](/ui-components/badge)** — Rótulos e tags de status.
- **[Avatar](/ui-components/avatar)** — Exibição de foto ou iniciais de usuários/clientes.
- **[Skeleton](/ui-components/skeleton)** — Placeholders visuais para estados de carregamento.
- **[Separator](/ui-components/separator)** — Divisores visuais horizontais e verticais.
- **[ScrollArea](/ui-components/scroll-area)** — Área de rolagem customizada.
- **[Tooltip](/ui-components/tooltip)** — Dicas flutuantes ao passar o mouse.
- **[DropdownMenu](/ui-components/dropdown-menu)** — Menus de contexto e ações rápidas.
- **[Loading](/ui-components/loading)** — Spinners e indicadores de progresso.
- **[PageLoader](/ui-components/page-loader)** — Tela cheia de carregamento de página.
- **[StatePanel](/ui-components/state-panel)** — Painel de estados (vazio, erro, sucesso).
