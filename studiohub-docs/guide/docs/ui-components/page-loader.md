# PageLoader

Indicador de carregamento para página inteira.

**Arquivo:** `src/ui-components/ui/page-loader.tsx`

```tsx
<PageLoader />
```

Usado durante lazy loading de páginas (React.lazy + Suspense). Ocupa a tela toda com um spinner centralizado.
