# Separator

Linha divisória horizontal ou vertical. Baseado em `@radix-ui/react-separator`.

**Arquivo:** `src/ui-components/ui/separator.tsx`

```tsx
<Separator />
<Separator orientation="vertical" className="h-full mx-2" />
```

- `orientation`: `horizontal` (padrão) | `vertical`
- Classes Tailwind para margem/altura
